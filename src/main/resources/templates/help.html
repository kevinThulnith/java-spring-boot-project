üìä Dashboard Overview

Your dashboard provides a comprehensive overview of your business performance with real-time statistics and quick access to important functions.

Statistics Cards

  • Total Customers: Shows your customer count
  • Total Items: Displays inventory count
  • Active Items: Items currently available
  • Low Stock Items: Items needing restocking
  • Bills: Total, paid, and draft bills

Navigation Features

  • Theme Toggle: Switch between light/dark modes
  • User Welcome: Shows current logged-in user
  • Quick Links: Access all main sections easily
  • Logout: Secure session termination

üì¶ Item Management

Adding Items:

  • Navigate to Items section
  • Click "Add New Item"
  • Enter item details (name, description, price, stock)
  • Set item status (ACTIVE/INACTIVE)

  • Monitor stock levels from dashboard
  • Update quantities as needed
  • Get alerts for low stock items
  • Activate/deactivate items as needed

üßæ Bill Management

Creating Bills:

  • Select customer from dropdown
  • Add items to the bill
  • System automatically calculates totals
  • Save as draft or mark as paid

Bill Status:

  • DRAFT: Pending completion
  • PAID: Payment received
  • Edit bills before marking as paid
  • View bill history and details

üîß Troubleshooting

Common Issues & Solutions

üö´ Cannot login to the system

  • Check your username and password
  • Ensure Caps Lock is not enabled
  • Clear your browser cache and cookies
  • Contact administrator if issue persists

üìä Dashboard not loading correctly

  • Refresh the page (F5 or Ctrl+R)
  • Check your internet connection
  • Try a different browser
  • Disable browser extensions temporarily

üíæ Data not saving

  • Ensure all required fields are filled
  • Check for error messages
  • Verify your session hasn't expired
  • Try saving again after a few moments

üé® Theme/Display issues

  • Try toggling between light and dark themes
  • Clear browser cache
  • Zoom level should be at 100%
  • Update your browser to the latest version

Best Practices

  • Always log out when finished to maintain security
  • Regularly backup important data
  • Keep your browser updated for best performance
  • Use strong, unique passwords
  • Report any suspicious activity immediately
